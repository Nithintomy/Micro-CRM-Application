``Micro-CRM Frontend``

``This is the frontend for the Micro-CRM application.``

``It is a simple React app used to test authentication, tenant isolation, and basic contact management.``

``Requirements``

Node.js(v20+)

npm

``Setup``

1. Install dependencies
npm install

2. Start the frontend
npm run dev


``The app will start on the default Vite port (usually 5173)``

``Backend connection``

``The frontend expects the backend to be running on:``

http://localhost:3001


``Make sure the backend server is started before using the frontend.``

``Features``

Login and logout

View contacts

Create, edit, and delete contacts

Access is restricted based on authentication
