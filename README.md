
``Micro-CRM Application``

``This repository contains a full-stack Micro-CRM application with a separate Backend and Frontend, each having its own detailed README.``

``Prerequisites``

Node.js (v18+)

npm

PostgreSQL (for backend)

``One Time Setup``

Install dependencies for both applications:

npm install


``This installs the root dependencies (used to run both apps together).``


Run Backend & Frontend Together (Single Command)

From the project root, run:

npm run dev


``This will start:``

``Backend - http://localhost:3001``

``Frontend (UX) - http://localhost:5173``

Both services run in parallel using a single command.

``For detailed setup, environment variables, database migrations, and tests, refer to:``
